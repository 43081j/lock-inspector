## lockcheck

A tiny utility to analyze package-lock.json files for potential problems,
inconsistencies and security issues.

### Install

```
$ npm i -D lockcheck
```

### Use

```
$ npx lockcheck
```

### Options

```
  -d, --dir <path>  Path to directory containing lock file. (default: ".")
  -h, --help        output usage information
```
